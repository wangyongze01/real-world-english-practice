# Real-World English Practice 使用说明

这个 skill 用来通过真实问题模拟练英语。每次练习都应该有三个核心要素：

- 场景：要解决什么实际问题
- 训练焦点：这次重点练什么语言能力
- 模式：角色扮演、专项句型、写作、改写、跟读风格或挑战
- 纠错风格：轻纠错、平衡纠错、严格纠错或教练模式
- 长度：短练习、标准练习或挑战练习

## 训练焦点

每次练习建议只选一个重点。这样反馈会更集中。

| 焦点 | 适合练什么 | 示例请求 |
|---|---|---|
| Comprehensive | 全面练习：意思、语法、词汇、语气、流畅度 | 本轮全面练习 |
| Tense | 时态、时间线、过去/现在/将来叙述 | 本轮重点练时态 |
| Collocations | 固定搭配、动词名词搭配、介词搭配 | 本轮重点练固定搭配 |
| Idiomatic expression | 地道表达、少一点中式英文 | 本轮重点练地道表达 |
| Conditionals/subjunctive | 条件句、假设、委婉请求、虚拟语气 | 本轮重点练虚拟语气 |
| Tone and politeness | 礼貌、商务语气、坚定但不冒犯 | 本轮重点练礼貌和语气 |
| Fluency and structure | 组织表达、解释原因、总结、转折 | 本轮重点练表达结构 |
| Pronunciation-oriented phrasing | 口语分块、重音、适合朗读的自然短句 | 本轮重点练口语节奏 |
| Writing clarity | 写作清晰度、语气、结构和行动导向 | 本轮重点练写作清晰度 |

## 训练模式

| 模式 | 适合练什么 | 示例请求 |
|---|---|---|
| Warm-up | 3-5 轮快速开口 | 今天先做一个热身练习 |
| Role-play | 标准真实场景对话 | 用角色扮演模式练酒店换房 |
| Drill | 集中练一个句型或语言点 | 用 drill 模式练 polite requests |
| Rewrite | 把句子或短消息改自然 | 帮我把这句话改得更自然 |
| Writing | 邮件、消息、投诉、follow-up | 帮我写一封投诉邮件 |
| Shadowing-style | 跟读、口语节奏、分块 | 给我适合跟读的自然表达 |
| Challenge | 拒绝、误解、谈判、时间压力 | 用挑战模式练客服退款 |

## 纠错风格

| 风格 | 适合什么时候用 |
|---|---|
| 轻纠错 | 想保持开口流畅，不想被频繁打断 |
| 平衡纠错 | 默认选择，每轮纠 1-3 个关键点 |
| 严格纠错 | 想细抠语法、搭配、语气和结构 |
| 教练模式 | 想知道规律，并马上做一个小练习巩固 |

## 练习长度

| 模式 | 轮数 | 适合什么时候用 |
|---|---:|---|
| 短练习 | 5 轮 | 热身、时间少、只练一个小点 |
| 标准练习 | 10 轮 | 默认选择，适合完整解决一个问题 |
| 挑战练习 | 15 轮 | 加入拒绝、误解、谈判、时间压力 |

## 常用场景

可以从这些场景开始：

- 酒店：要求换房、投诉噪音、协商升级费
- 客服：申请退款、解释商品损坏、争取补偿
- 工作：解释项目延期、同步进度、提出新计划
- 面试：回答行为问题、介绍项目经历、反问岗位信息
- 旅行：行李丢失、航班延误、改签
- 医疗：描述症状、预约检查、询问用药
- 租房：要求维修、沟通押金、处理邻居噪音
- 餐厅：点错菜、过敏提醒、投诉服务

## 高质量提示词

### 1. 时态专项

```text
用 $real-world-english-practice 陪我练一个客服退款场景。
我的水平是中级。
本轮重点练 Tense：我要练清楚描述过去发生了什么、现在的问题是什么、希望接下来怎么处理。
标准练习 10 轮。
练完后复盘并归档到飞书文档。
```

### 2. 固定搭配专项

```text
用 $real-world-english-practice 陪我练一个工作延期沟通场景。
我的水平是中级。
本轮重点练 Collocations，帮我纠正不自然的词语搭配。
标准练习 10 轮。
```

### 3. 地道表达专项

```text
用 $real-world-english-practice 陪我练一个酒店投诉场景。
我的水平是中级。
本轮重点练 Idiomatic expression。先保证我能表达清楚，再帮我升级成更自然的说法。
短练习 5 轮。
```

### 4. 虚拟语气和条件句专项

```text
用 $real-world-english-practice 陪我练一个协商退款场景。
我的水平是中高级。
本轮重点练 Conditionals/subjunctive，尤其是 if, would, could, might 的用法。
挑战练习 15 轮。
```

### 5. 商务语气专项

```text
用 $real-world-english-practice 陪我练一个和同事沟通项目延期的场景。
我的水平是中级。
本轮重点练 Tone and politeness：表达要专业、礼貌、但不能太软。
标准练习 10 轮。
```

### 6. 写作修改

```text
用 $real-world-english-practice 帮我改一封英文投诉邮件。
我的水平是中级。
本轮重点练 Writing clarity 和 Tone and politeness。
纠错风格是教练模式。
我会先发我的草稿，请先保留我的原意，再给我一个更自然、更礼貌但坚定的版本。
```

### 7. 句型专项 drill

```text
用 $real-world-english-practice 带我做一个 drill。
我的水平是 B1。
重点练 Would it be possible to...? 和 Is there any way you could...?
场景围绕酒店、客服和工作沟通各来几轮。
每次我回答后给一句更自然的版本。
```

## 复盘会包含什么

每次练完后，复盘建议包含：

```markdown
## 日期 + 场景名称

### Goal
本轮任务目标。

### Profile
水平、模式和纠错风格。

### Focus
本轮训练焦点。

### Outcome
任务完成情况。

### Focus Feedback
针对训练焦点的反馈。

### Top Mistakes
最值得改的 2-4 个错误。

### Reusable Phrases
下次可以直接复用的句子。

### Natural Upgrades
我的原句 -> 更自然表达。

### Next Practice
下一次建议练什么。
```

## 推荐使用节奏

- 每天 1 次短练习或标准练习。
- 每周选 1 个训练焦点重复练 3-5 次。
- 每周末看一次飞书文档里的 Top Mistakes。
- 同一个场景可以换不同焦点重复练，比如酒店投诉可以分别练时态、固定搭配、礼貌语气和地道表达。
